FTL

Faster Than Light — a safety-first, auditable systems programming language focused on explicit behavior and predictable compilation.

GitHub v0.0.1

What is FTL?

FTL (Faster Than Light) is an experimental systems programming language designed with correctness, safety, and auditability as first-class goals.

The language avoids hidden behavior and compiler magic, favoring explicit semantics and transparent compilation stages.

Design Principles

  • Auditability over magic
  • Explicit, predictable behavior
  • Minimal preprocessing
  • Clear compiler boundaries
  • Well-defined semantics

Compiler Architecture 

Source Code
    ↓
Corona    (preprocessing / includes)
    ↓
Borealis  (IR generation)
    ↓
Quark     (optimization)
    ↓
Photon    (bytecode generation, planned)

Project Status

FTL is in early development (v0.0.x). Breaking changes are expected.

Current development focuses on compiler correctness, safety invariants, and formal specification.
